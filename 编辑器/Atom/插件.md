# Atom 插件介绍

## autoclose-html

> 看名字就知道了. `auto`自动,`close`关闭,`html`html 标签

自动关闭 html 标签

## autocomplete-paths

> 自动填充加路径提醒

## codelf

> 调用 codelf

## file-icons

> 侧边栏文件的 icon

## color-picker

> 取色器

## autoprefixer

> 补充 css 前缀

## atom-ternjs

> JS 代码智能提示补全

- browser: completion for vanilla js (optional)
- ecma5: es5 (optional)
- ecma6: es6 (optional)
- jquery: completion for jQuery (optional) – 这个可以有，JQ 也支持

## javascript-snippets

> JS 代码片段补全

这款插件就比较重量级了，用熟悉了写原生 JS 的效率要提升很多。而且，不仅支持 JS 还包含了 nodejs snippet。
